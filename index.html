Preguntas frecuentes

Dudas comunes sobre gestión de flota y mapa de patio

¿MapGestion sirve para gestión de flota?expand_more
Sí. MapGestion permite gestionar flotas, unidades, estados, usuarios, plazas, incidencias y movimientos desde una plataforma web con mapa de patio.
¿Puedo usarlo como mapa de estacionamiento o mapa de patio?expand_more
Sí. Puedes configurar zonas, cajones, patios, plazas o estacionamientos para ubicar unidades y ver disponibilidad, mantenimiento, limpieza o resguardo.
¿Sirve para arrendadoras de autos o camiones?expand_more
Sí. MapGestion ayuda a arrendadoras a controlar disponibilidad, contratos, estados de unidad, preparación, incidencias y movimientos por sucursal.
¿Es necesario instalar algún software especial?expand_more
No. MapGestion funciona desde navegador web y también está pensado para operadores que trabajan desde celular en patio.
¿Cuánto tarda implementar MapGestion?expand_more
Depende del tamaño del patio y la cantidad de unidades, pero el mapeo inicial y la configuración suelen tomar menos de una semana.
¿Qué pasa si tengo varias sucursales o patios?expand_more
Puedes operar por plaza y limitar accesos por rol. El plan Regional y Corporativo permiten visibilidad multi-plaza para gerentes o dirección.
